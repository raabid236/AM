Nearest_distance:-

opt_bruteforce.m: runs brute force method
opt_SA.m: runs simulated annealing method
opt_pso.m: runs particle swarm optimization method
costfunction.m: compute the distance based cost function
costfunction_SA.m: compute the distance based cost function for opt_SA.m



Volume_based:-

algo.m: runs the optimization algorithm
volum.m: computes the volume based cost function
trans6.m: calculates the transformation matrix
bun.mat: point cloud of a bunny
mySave.mat: point cloud of a sprocket

Note: To  change the dataset, simply load the other .mat file in the code (commented)
